## Getting Started

### Installation

Install the dependencies:

```bash
pnpm install
```

### Development

Start the development server with HMR:

```bash
pnpm dev
```

The application will be available at `http://localhost:2000`.

## Building for Production

Create a production build:

```bash
pnpm build
```
